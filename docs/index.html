法人保険 ／ 経理処理支援

仕訳ジェネレーター

入力項目に答えると、仕訳(振替伝票)とそのまま送れるメール文を作成します。計算は固定ロジックで行うため、再現・検算が可能です。

INPUT

入力項目

養老ハーフタックス(法基通9-3-4(3))。普遍的加入は福利厚生費、特定者のみは給与。いずれも損金算入率は1/2(資産計上50%・自動設定)。
※この1/2ルールは平準払が前提です。一時払・短期払(払込期間＜保険期間)の場合は前払保険料の期間按分が必要で、本ツールの対象外です。
新通達の区分:最高解約返戻率 50%以下=全額損金／50%超70%以下=40%資産／70%超85%以下=60%資産。1/2損金は旧長期平準定期・養老ハーフタックス等。
※本ツールは前半の資産計上期間を対象とします。後半の取崩期間(前半で積んだ前払保険料を残余期間で均等取崩し)や85%超の年別スケジュールは自動計算しません。取崩開始後は「全額損金＋取崩額」を手動で組むか、解約時は帳簿の資産計上簿価を直接入力してください。
契約日で適用通達（2019/7/8以降＝新通達／以前＝旧通達）を自動判定し、下の追加入力で資産計上割合を確定します。空欄なら上の手動選択のままです。判定は前半の資産計上期間が対象で、後半取崩し・85%超の年別推移は含みません。
⚠ 推計値です。可能な限り帳簿の資産計上簿価をご使用ください。
毎年の端数（奇数保険料×割合）は資産側へ丸めるため、実際の資産計上簿価は「払込累計×割合」の推計と数円〜ずれます（例：1/2損金・奇数保険料なら年1円ずつ資産が過大＝10年で約10円）。解約・満期は実際に計上した簿価を取り崩すのが正しく、推計だと雑収入／雑損失が端数分ずれます。後半期間の取崩し開始後や契約者配当がある場合はさらに乖離します。
解約日と着金日が異なる場合に入力。仕訳は着金日付で計上し、メールには両方を記載します。空欄なら解約日付で計上します。

複数契約を一括処理します。取引区分と共通設定を選び、契約一覧を貼り付けてください。

⚠ 経理区分は全契約に一律適用されます。定期・第三分野で一の被保険者・当該事業年度の年換算保険料が30万円以下の契約は全額損金の特例対象（同種複数契約は合算判定／法基通9-3-5(注)2）。閾値を跨ぐ契約が混在する場合は、区分ごとに分けて生成してください。
⚠ 複数契約では「資産計上残高」の入力を強く推奨します。
払込累計からの推計は、契約ごとに毎年の端数が資産側へ丸められて累積するため（奇数保険料×割合で1契約あたり年1円ずつ）、実際の資産計上簿価と契約ごとにずれます。件数が多いほど誤差が積み上がり、保険料実績から正確に割り戻すのは困難です。実際に計上した簿価（帳簿残高）を2列目に入力してください。
着金日(予定)は全契約共通とし、契約ごとには表示せず合計欄に記載します。
MAIL

メール用の宛先(任意)

OUTPUT

仕訳とメール文

＃ 左の項目を入力し「仕訳を作成」を押してください。
本ツールの仕訳は入力値に基づく機械計算です。最高解約返戻率85%超の複雑な資産計上スケジュールや、後半期間の取崩し計算は自動化していません(該当時は帳簿の資産計上残高を直接入力してください)。最終的な経理・税務処理は顧問税理士の確認のうえ確定してください。
